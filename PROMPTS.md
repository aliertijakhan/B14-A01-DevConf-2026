# AI Prompt(s)

## Prompt 1

Design a modern FAQ section for a DevConf 2026 landing page using only HTML5 and CSS3. The section should include a heading, a short introductory paragraph, and four frequently asked questions with expandable answers using the HTML5 `<details>` and `<summary>` elements. Arrange the FAQ items in a two-column CSS Grid layout. Use only semantic HTML and plain CSS (no JavaScript, Bootstrap, Tailwind, or other frameworks). Make the design clean, professional, and visually consistent with a modern developer conference website.
